  • Install Git.
  • Right click on the file and select Git Bash Here.
  • Start working with the git by typing commands : $ git init
  • Add all files with the command : $ git add .
  • Register in the repository of the gate with a suitable and complete message : $ git commit -m "Your message"
  • If this is your first time using Git, you need to create an account : $ git config -- user.name "***"
  • Enter your email : $ git config -- user.email "web@gmail.com
  • We create a repository through the GitHub account and copy it here : $ git remote add origin "Your address"
  • We transfer the project to the GitHub : $ git push -u origin master
